# Cidadania Digital e Inteligência Artificial: O Impacto das Deepfakes e da Desinformação na Sociedade

## 🎯 Objetivo do Projeto
Este site foi desenvolvido como atividade de Recuperação Prática para o componente curricular de Educação Digital e IA (2026). O intuito do projeto é instruir a comunidade escolar de forma didática sobre o conceito de deepfakes, os perigos inerentes à desinformação automatizada gerada por algoritmos de inteligência artificial e quais as diretrizes funcionais para a identificação de mídias fraudulentas.

## 💻 Recursos Tecnológicos Implementados
- **HTML5:** Utilização rigorosa de elementos semânticos estruturais (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<form>`, `<footer>`) visando a acessibilidade e organização do código.
- **CSS3 Avançado:** Aplicação de Layout Flexbox para total alinhamento e flexibilidade dos cards informativos, variáveis CSS para fácil manutenção das cores e estilização com estética futurista baseada em Neomorfismo/Neon.
- **Responsividade Automatizada:** Inclusão de Media Queries direcionadas para garantir usabilidade em telas de celulares e desktops.
- **Vanilla JavaScript:** Construção sem bibliotecas externas de um alternador de tema dinâmico, um motor de Quiz iterativo para validação de conhecimento e um manipulador do DOM para validação do formulário de denúncia escolar.

## 🤖 Declaração de Uso de Inteligência Artificial (Ética)
Em total conformidade com os critérios do projeto, declaro que utilizei a Inteligência Artificial Gemini (da Google) para auxiliar na estruturação lógica do escopo e geração de textos técnicos de conscientização digital.
